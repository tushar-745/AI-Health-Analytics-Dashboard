# AI Health Analytics Dashboard

An interactive AI-powered healthcare analytics dashboard built with **Python, Streamlit, Plotly, and Machine Learning** to predict Heart Disease and Diabetes risk while providing hospital and patient analytics through interactive visualizations.

# 🏥 AI Health Analytics Dashboard

This repository contains the **AI Health Analytics Dashboard** developed using **Python, Streamlit, Plotly, Scikit-learn, and Pandas**.

The application combines **machine learning models** with **interactive healthcare analytics** to help visualize patient data, monitor hospital performance, and predict the likelihood of **Heart Disease** and **Diabetes**.

---

## 📂 Project Files

- **app.py** → Main Streamlit application.
- **data/** → Patient, Diabetes, and Heart Disease datasets.
- **models/** → Trained Machine Learning models and scalers.
- **Preview Images** → Dashboard screenshots.

# 🚀 Dashboard Features

### 📊 Executive Dashboard
- Interactive healthcare overview dashboard.
- Patient status and department analysis.
- Monthly admission trends.
- Gender-wise age distribution.
- Hospital KPI cards.

### ❤️ Heart Disease Prediction
- Machine Learning-based Heart Disease prediction.
- Interactive patient input form.
- Risk probability gauge.
- Feature importance visualization.
- Disease prevalence analysis.

### 🩸 Diabetes Prediction
- Random Forest-based Diabetes prediction.
- Interactive medical parameter inputs.
- Risk probability visualization.
- Glucose vs BMI analysis.
- Feature importance chart.

### 🏥 Patient Analytics
- Department-wise treatment cost analysis.
- Patient satisfaction monitoring.
- Length of Stay distribution.
- BMI vs Cholesterol analysis.
- Dynamic filters for Department, Gender, and Status.
- Patient records table.

### 📈 Model Performance
- Diabetes & Heart Disease model accuracy.
- ROC-AUC performance metrics.
- Feature importance comparison.
- Dataset distribution analysis.
- Machine Learning model summary.

# 🤖 Machine Learning Models

### Diabetes Prediction
- Random Forest Classifier
- Accuracy: **85.5%**
- ROC-AUC Score: **0.924**

### Heart Disease Prediction
- Gradient Boosting Classifier
- Accuracy: **84.0%**
- ROC-AUC Score: **0.906**

# 💡 How to Run

1. Clone this repository.

```bash
git clone https://github.com/your-username/AI-Health-Analytics-Dashboard.git
```

2. Navigate to the project folder.

```bash
cd AI-Health-Analytics-Dashboard
```

3. Install dependencies.

```bash
pip install -r requirements.txt
```

4. Run the Streamlit application.

```bash
streamlit run app.py
```
